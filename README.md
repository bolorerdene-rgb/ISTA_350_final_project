Project Title: Car Sales Data Analysis Using Web Scraping

Group Members:
Bolor-Erdene Altanmunkh

Project Description:
This project will analyze automobile sales data by scraping structured tables from Wikipedia. The primary dataset will be sourced from the List of automobile sales by model page, which contains information on car models, manufacturers, production years, and total units sold. A secondary dataset may be used from the List of best selling automobiles page to support and validate the analysis.

The goal of this project is to explore relationships between different factors that influence car sales performance. Specifically, I plan to investigate how production duration relates to total sales, compare performance across manufacturers, and identify the most successful car models globally.
